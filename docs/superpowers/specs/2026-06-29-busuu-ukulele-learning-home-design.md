# Busuu 风格尤克里里学习主页设计

## 背景

项目目录当前为空，仅包含 Git 仓库。目标是创建一个可直接打开的静态网站基础框架，用 Busuu 参考稿里的视觉语言承载“尤克里里教学助手”这个原创教学产品。

本设计借鉴 Busuu 的界面气质：紫蓝渐变首屏、轻快插画感、圆形课程入口、淡蓝白画布、白色卡片和绿色胶囊主按钮。实现时不复制 Busuu 的 Logo、品牌素材、文案或真实页面结构，所有内容与图形都改为尤克里里教学主题。

## 用户目标

用户打开首页后，应能立刻知道自己可以开始学习尤克里里，并快速进入课程、练习、曲谱或今日学习任务。页面更偏“学习入口”，不是单纯招生落地页。

## 成功标准

- 首页视觉上接近参考稿的轻快、圆润、国际化学习产品气质。
- 首屏明确表达“今天开始学尤克里里”，并提供继续学习和选择课程两个入口。
- 课程分类以圆形入口呈现，形成类似 Busuu 语言选择卡片的记忆点。
- 内容区包含今日推荐、学习进度、热门曲谱、练习计划等基础模块。
- 页面在桌面和手机宽度都可阅读、无明显重叠、按钮和卡片尺寸稳定。
- 初版可作为静态站点直接打开，无需安装依赖。

## 页面架构

### 顶部导航

- 白色固定导航栏。
- 左侧为原创 Logo 锁定区：圆角蓝色图标加“尤克里里教学助手”文字。
- 右侧导航项：课程、练习、曲谱、登录。
- 主操作按钮为绿色胶囊按钮“开始学习”。

### 首屏学习入口

- 使用全宽紫蓝渐变背景。
- 左侧文案：
  - 主标题：“今天，从一首歌开始学尤克里里”
  - 副文案：强调零基础、和弦、节奏和弹唱路径。
  - 主按钮：“继续学习”
  - 次按钮：“选择课程”
- 右侧为原创 CSS 插画区域：
  - 尤克里里主体。
  - 圆形音符气泡。
  - 简化课程徽章。
  - 轻微世界地图/五线谱式背景线条，保持学习产品感。

### 圆形课程入口

标题为“今天想练什么？”

课程入口：
- 零基础入门
- 常用和弦
- 扫弦节奏
- 弹唱练习
- 曲谱库
- 每日练习

每个入口使用白色圆形卡片承载一个简洁图形符号，下面放课程名称。圆形是视觉系统的关键特征。

### 学习内容区

使用淡蓝白背景和白色卡片：
- 今日推荐：展示一个可继续学习的课程。
- 学习进度：展示进度条和已完成章节。
- 热门曲谱：展示 3 首示例曲谱。
- 练习计划：展示每日 15 分钟练习拆分。

### 页脚

- 延续淡蓝白背景。
- 放一句产品说明和简单链接。
- 提供“显示语言”下拉框，为后续多语言扩展留位置。

## 视觉系统

### 颜色

- 页面背景：`#f2f7fd`
- 卡片背景：`#ffffff`
- 主文字：`#252b2f`
- 次文字：`#666e7e`
- 品牌蓝：`#116eee`
- 主按钮绿：`#11ee92`
- 首屏渐变：`linear-gradient(90deg, #3b1e90 0%, #5a3cc4 30%, #3a6ef0 100%)`
- 边框：`#d6dee6`
- 轻阴影：`rgba(0, 0, 0, 0.1) 0px 1px 2px 0px`

### 字体

使用本地可用的圆润无衬线字体栈，优先 `Poppins`、`Nunito`，再回落到系统字体。中文文本保持清晰可读，标题使用较粗字重。

### 形状

- 主按钮：45px 胶囊圆角。
- 课程入口：9999px 圆形。
- 内容卡片：16px 圆角。
- 输入框和下拉框：5px 圆角。

## 交互

- 导航链接滚动到对应区域。
- “继续学习”按钮定位到今日推荐卡片。
- “选择课程”按钮定位到课程入口。
- 课程圆形入口 hover 时轻微上浮。
- 移动端导航折叠为更紧凑的横向布局，不做复杂菜单。

## 技术方案

初版使用静态文件：
- `index.html`：页面结构和内容。
- `styles.css`：视觉系统、响应式布局和插画样式。
- `script.js`：平滑滚动、课程入口点击反馈等轻量交互。

不引入 React、构建工具或外部依赖。这样可以直接双击打开，也方便后续迁移到 Vite/React。

## 响应式规则

- 桌面端首屏为左右两列。
- 平板端减少间距，插画缩小。
- 手机端首屏改为单列，文案在上、插画在下。
- 课程入口使用自适应网格，最小卡片宽度保持可点击。
- 所有按钮文字不得溢出，长文本自动换行。

## 验证方式

- 直接打开 `index.html` 检查静态页面。
- 使用本地静态服务器或浏览器预览桌面和移动宽度。
- 检查导航跳转、按钮跳转和课程入口反馈。
- 目视检查首屏、课程区和内容卡片是否符合 Busuu 风格参考。
